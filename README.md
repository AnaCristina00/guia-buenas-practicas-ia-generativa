## Descargar versión sin conexión de la guía de buenas prácticas para el uso responsable de Inteligencia Artificial Generativa

Debido al tamaño del archivo, la versión descargable de la guía se encuentra disponible en el siguiente enlace:

https://drive.google.com/file/d/1SNSVsYsXD3rDNf_mv4UvzeEU0Z9EtF9x/view?usp=drive_link

Pasos para visualizarla:

1. Descargar el archivo .zip
2. Descomprimir el archivo
3. Abrir la carpeta
4. Ejecutar el archivo **genially.html**

Dentro de la carpeta también se encuentra un archivo .pdf llamado **Guía para presentar sin conexión.pdf**, en él están las instrucciones para poder revisar la guía sin necesidad de conexión a internet.
